# Azure Core Concepts

## Subscription

A subscription represents a billing and management boundary in Azure.

Resources deployed within a subscription share billing and governance controls.

## Resource Group

A resource group is a logical container used to organize related resources.

Resources inside a resource group often share lifecycle and management patterns.

## Region

Azure regions represent geographic data center locations where resources are deployed.

Example region used in this lab:

East US

## Budget

Budgets help control spending by providing alerts when cost thresholds are reached.

This lab uses a monthly budget for cost control.
