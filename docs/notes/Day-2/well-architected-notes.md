# Azure Well-Architected Framework

The Well-Architected Framework provides principles for designing reliable cloud systems.

The framework includes five pillars.

## Reliability

Systems should remain functional during failures and recover quickly.

## Security

Workloads should protect data, systems, and identities.

## Cost Optimization

Architectures should control and optimize cloud spending.

## Operational Excellence

Systems should be easy to monitor, operate, and improve.

## Performance Efficiency

Systems should scale and perform efficiently under varying workloads.
